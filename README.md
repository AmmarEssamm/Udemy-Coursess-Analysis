# Udemy-Coursess-Analysis

### Overview
Analyzed Udemy course data to understand pricing strategies, ratings,
subscriber behavior, and course popularity.

### Tools
- Python
- pandas, NumPy
- matplotlib, seaborn

### Key Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insight Generation

### Outcome
Delivered insights into course performance and learner preferences,
demonstrating strong analytical and visualization skills.
